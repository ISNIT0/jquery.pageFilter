Easily add a filter to your web page.

Easy to use... No special Clases or Functions, just 100% reason to remember the name.
Works with any type of element, UL is only an example.

Using it.

Code:

<input class='pageFilter'>

<ul class="tofilter">
   <li>Volvo</li>
   <li>Saab</li>
   <li>Opel</li>
   <li>Audi</li>
</ul>



$('.pageFilter').pageFilter('.tofilter')
